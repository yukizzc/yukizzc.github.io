matplotlib

Created2024-07-26|Updated2026-03-28|computer
|Post Views:

Matplotlib 是 Python 的 2D 绘图库，用于生成高质量的图表（如折线图、散点图、柱状图等）

属性

线条相关属性

plt.plot() 本身支持许多参数，用于控制线条样式、颜色、宽度等：

  • color：线条颜色（例如 ‘blue’、‘r’、#FF5733’）
  • linewidth 或 lw：线条宽度（例如 2.5）
  • linestyle 或 ls：线条样式（例如 ‘-’ 实线、‘–’ 虚线、‘:’ 点线）
  • marker：数据点的标记样式（例如 ‘o’ 圆点、‘s’ 方块、‘^’ 三角形）
  • markersize 或 ms：标记大小

坐标轴相关属性

可以通过 plt 或 Axes 对象调整坐标轴的宽度、范围、刻度等：

  • plt.xlim(min, max) / plt.ylim(min, max)：设置 x 和 y 轴的显示范围
  • plt.axis([xmin, xmax, ymin, ymax])：同时设置 x 和 y 轴范围
  • plt.tick_params()：调整刻度的样式（长度、宽度、方向等）
  • plt.xticks() / plt.yticks()：自定义刻度位置和标签

调整坐标轴线宽度（Spines）

刻度显示数量和样式

  • plt.locator_params()：控制刻度数量
  • plt.tick_params()：调整刻度线的长度、宽度、方向等

网格线

可以用 plt.grid() 添加网格线，并调整样式：

  • plt.grid(True)：显示网格
  • color、linestyle、linewidth：自定义网格样式

图表整体样式

  • plt.figure(figsize=(width, height))：设置图表大小（单位：英寸）

  • plt.style.use(‘style_name’)：使用预定义样式（如 ‘ggplot’、‘seaborn’）

其他常用属性

  • plt.legend()：添加图例（需在 plt.plot() 中指定 label）
  • plt.text(x, y, ‘text’)：在图中添加文本
  • plt.annotate(‘text’, xy=(x, y))：添加带箭头的标注

常用图表

总结表

图表类型 函数 适用场景
折线图 plt.plot() 趋势变化
散点图 plt.scatter() 数据点关系
柱状图 plt.bar() 类别比较
直方图 plt.hist() 数据分布
饼图 plt.pie() 比例分配
面积图 plt.fill_between() 累积变化
箱线图 plt.boxplot() 统计分布
热力图 plt.imshow() 数据强度/相关性
多子图 plt.subplots() 多图对比

折线图

用途：展示数据随时间或其他连续变量的变化趋势

散点图

用途：显示两个变量之间的关系，适合离散数据点

柱状图

用途：比较不同类别的数据大小

直方图

用途：展示数据的分布情况

饼图

用途：表示各部分占总体的比例

面积图

用途：展示数据的累积变化，通常用于时间序列

箱线图

用途：展示数据的分布、离群点和中位数等统计特征

热力图

用途：展示二维数据的强度或相关性

函数：需要结合 plt.imshow() 或 seaborn（更推荐）

多图组合

matplotlib 提供了多种方法来创建多子图，主要通过 plt.subplots() 或 plt.subplot() 实现。可以选择将多个子图放在一起绘制（共享一个画布）或分开绘制（独立窗口或画布）

叠加图形

共享画布

**plt.subplots() **

  • 描述：一次性创建整个画布和多个子图，返回 Figure 和 Axes 对象
  • 参数
    • nrows：行数
    • ncols：列数
    • figsize：画布大小

plt.subplot()

  • 描述：逐个添加子图，适合动态调整或手动布局
  • 参数：plt.subplot(nrows, ncols, index)，index 从 1 开始
  • 特点：更灵活，但需要手动指定每个子图的位置

独立画布

方法：多次调用 plt.figure() 创建多个独立的 Figure 对象，每次绘图后调用 plt.show()

特点：

  • 每个图表有独立的画布，显示在单独的窗口或分开保存
  • 适合需要独立查看或保存的场景

常用场景：不需要直接对比，或希望分开展示

共享坐标轴

用途：多个子图共享 x 或 y 轴，适合对比数据

参数：sharex 或 sharey

不规则布局

用途：创建非均匀的子图布局

方法：matplotlib.gridspec.GridSpec

总结

方式 方法 特点 适用场景
放在一起 plt.subplots() 一次性创建，共享画布 对比多个图表
放在一起 plt.subplot() 逐个添加，灵活布局 动态调整布局
分开绘制 plt.figure() + show() 独立画布，单独显示 独立查看或保存
高级布局 GridSpec 不规则布局，高度自定义 复杂布局需求
