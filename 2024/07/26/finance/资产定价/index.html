资产定价基础

理论基础

资本资产定价模型（Capital Asset Pricing Model，CAPM）由威廉·夏普（William Sharpe）、约翰·林特纳（John Lintner）等学者在1960年代独立提出，是现代金融学的基石之一。该模型回答了核心问题：资产的预期收益率应该如何决定？

CAPM的核心思想：

  • 资产的预期收益由两部分构成：无风险收益 + 风险溢价
  • 只有系统性风险（市场风险）才应该获得风险补偿
  • 非系统性风险可以通过分散化投资消除，不应当获得额外收益

模型公式

E(Ri)=Rf+βi×[E(Rm)−Rf]E(R_i) = R_f + \beta_i \times [E(R_m) - R_f] E(Ri​)=Rf​+βi​×[E(Rm​)−Rf​]

其中：

  • E(Ri)E(R_i)E(Ri​)：资产i的预期收益率
  • RfR_fRf​：无风险利率（通常用国债收益率）
  • βi\beta_iβi​：资产i的贝塔系数，衡量资产对市场波动的敏感度
  • E(Rm)E(R_m)E(Rm​)：市场组合的预期收益率
  • E(Rm)−RfE(R_m) - R_fE(Rm​)−Rf​：市场风险溢价

Beta系数的含义

Beta系数是CAPM的核心，计算公式：

βi=Cov(Ri,Rm)Var(Rm)\beta_i = \frac{\text{Cov}(R_i, R_m)}{\text{Var}(R_m)} βi​=Var(Rm​)Cov(Ri​,Rm​)​

实际计算会用线性回归去计算系数得到的结果就是上面计算公式的结果

Beta的解读：

Beta值 含义 投资特征
β > 1.5 激进型 波动远大于市场，高风险高收益
1 < β < 1.5 进攻型 波动大于市场，牛市表现更好
β = 1 中性 与市场同步波动
0 < β < 1 防御型 波动小于市场，熊市相对抗跌
β = 0 无相关 与市场无关（如现金）
β < 0 反向型 与市场反向波动，对冲工具

核心假设

CAPM建立在以下假设之上：

  1. 投资者理性：投资者追求效用最大化，厌恶风险
  2. 市场有效：所有信息及时反映在价格中
  3. 无摩擦市场：无交易成本、无税收
  4. 同质预期：所有投资者对资产收益有相同的预期
  5. 可无限借贷：投资者可以按无风险利率借贷

资本成本（WACC）

什么是WACC？

  • 公司借钱（债务）和股东投钱（权益）都是有成本的
  • WACC就是这些成本的"加权平均"
  • 用途：作为折现率，用来计算公司值多少钱

权益资本成本（股东要求的回报）

公式：

re=rf+β×(rm−rf)r_e = r_f + \beta \times (r_m - r_f) re​=rf​+β×(rm​−rf​)

解释：

  • rfr_frf​：无风险利率（比如买国债的收益，稳赚的）
  • β\betaβ：股票比大盘波动多少（β=1.2表示比大盘波动大20%）
  • rm−rfr_m - r_frm​−rf​：市场风险溢价（投资股票比国债多赚的部分）

债务资本成本（借钱成本）

公式：

rdafter-tax=rd×(1−Tc)r_d^{\text{after-tax}} = r_d \times (1 - T_c) rdafter-tax​=rd​×(1−Tc​)

解释：

  • 借钱要付利息，但利息可以抵税
  • 实际成本 = 利率 × (1 - 税率)

WACC加权平均资本成本

公式：

WACC=EV×re+DV×rd×(1−Tc)\text{WACC} = \frac{E}{V} \times r_e + \frac{D}{V} \times r_d \times (1 - T_c) WACC=VE​×re​+VD​×rd​×(1−Tc​)

白话解释：

  • 公司价值 = 股东的钱(E) + 借的钱(D)
  • WACC = (股东占比 × 股东要求回报) + (债务占比 × 税后利息)

绝对估值法

绝对估值法概述

什么是绝对估值法？

绝对估值法是通过预测公司未来的现金流，并将其折现到当前时点来计算公司内在价值的方法。与相对估值法（如PE、PB）不同，绝对估值法不依赖市场可比公司，而是基于公司自身的财务预测。

核心思想

货币的时间价值：今天的1元钱比未来的1元钱更值钱，因为今天的钱可以投资获得收益。

基本公式：
​​

V0=∑t=1∞CFt(1+r)tV_0 = \sum_{t=1}^{\infty} \frac{CF_t}{(1+r)^t} V0​=t=1∑∞​(1+r)tCFt​​

其中：

  • V0V_0V0​：当前价值
  • CFtCF_tCFt​：第t期的现金流
  • rrr：折现率
  • ttt：时间期数

主要方法

方法 现金流类型 折现率 适用场景
DCF 自由现金流 WACC 适用于大多数公司
DDM 股利 权益成本 适用于稳定分红的公司
剩余收益 超额收益 权益成本 适用于账面价值重要的公司

DCF现金流折现模型

什么是自由现金流？

定义：自由现金流（Free Cash Flow，FCF）是公司在满足所有必要投资后，可以自由分配给股东和债权人的现金。

两种自由现金流：

1. 公司自由现金流（FCFF）

FCFF=EBIT×(1−T)+Depreciation−CapEx−ΔNWC\text{FCFF} = \text{EBIT} \times (1 - T) + \text{Depreciation} - \text{CapEx} - \Delta \text{NWC} FCFF=EBIT×(1−T)+Depreciation−CapEx−ΔNWC

  • EBITEBITEBIT：息税前利润
  • TTT：所得税税率
  • DepreciationDepreciationDepreciation：折旧与摊销
  • CapExCapExCapEx：资本性支出
  • ΔNWC\Delta NWCΔNWC：营运资本变动

2. 股权自由现金流（FCFE）

FCFE=FCFF−Interest×(1−T)+Net Borrowing\text{FCFE} = \text{FCFF} - \text{Interest} \times (1 - T) + \text{Net Borrowing} FCFE=FCFF−Interest×(1−T)+Net Borrowing

  • FCFEFCFEFCFE 为权益自由现金流（归属于股东的自由现金流）
  • FCFFFCFFFCFF 为企业自由现金流
  • InterestInterestInterest 为利息支出
  • TTT 为公司所得税税率
  • Interest×(1−T)Interest×(1−T)Interest×(1−T) 为税后利息支出（支付给债权人）
  • NetBorrowingNet BorrowingNetBorrowing 为净借款（新增债务减去偿还债务）

DCF模型的基本形式

两阶段模型是最常用的DCF形式：

V0=∑t=1nFCFt(1+WACC)t+TV(1+WACC)nV_0 = \sum_{t=1}^{n} \frac{FCF_t}{(1 + \text{WACC})^t} + \frac{TV}{(1 + \text{WACC})^n} V0​=t=1∑n​(1+WACC)tFCFt​​+(1+WACC)nTV​

其中：

  • V0V_0V0​：公司当前价值
  • FCFtFCF_tFCFt​：第t年的自由现金流
  • WACCWACCWACC：加权平均资本成本
  • TVTVTV：终值（Terminal Value）
  • nnn：预测期年数（通常5-10年）

终值的计算

终值代表预测期之后所有现金流的价值，主要有两种计算方法：

方法一：永续增长模型（最常用）

TV=FCFn×(1+g)WACC−gTV = \frac{FCF_n \times (1 + g)}{\text{WACC} - g} TV=WACC−gFCFn​×(1+g)​

其中：

  • ggg：永续增长率（通常为2%-4%，不超过GDP增长率）
  • WACC>gWACC > gWACC>g：必要条件

方法二：退出倍数法

TV=EBITDAn×MultipleTV = \text{EBITDA}_n \times \text{Multiple} TV=EBITDAn​×Multiple

其中：

  • MultipleMultipleMultiple：参考可比公司的EV/EBITDA倍数

DCF模型的核心假设

  1. 永续增长率：通常等于长期GDP增长率或通货膨胀率
  2. WACC：假设资本结构保持稳定
  3. 预测期长度：通常5-10年，直到公司达到稳定增长状态

剩余收益模型

基本概念

剩余收益模型认为公司价值等于账面价值加上未来超额收益的现值。超额收益是指超过权益资本成本的收益。

基本公式：
​​

V0=B0+∑t=1∞RIt(1+re)tV_0 = B_0 + \sum_{t=1}^{\infty} \frac{RI_t}{(1 + r_e)^t} V0​=B0​+t=1∑∞​(1+re​)tRIt​​

其中：

  • B0B_0B0​：当前每股账面价值
  • RItRI_tRIt​：第t期的每股剩余收益

剩余收益的计算

RIt=NIt−re×Bt−1RI_t = NI_t - r_e \times B_{t-1} RIt​=NIt​−re​×Bt−1​

或者：

RIt=Bt−1×(ROEt−re)RI_t = B_{t-1} \times (ROE_t - r_e) RIt​=Bt−1​×(ROEt​−re​)

其中：

  • NItNI_tNIt​：第t期净利润
  • ROEtROE_tROEt​：第t期净资产收益率
  • Bt−1B_{t-1}Bt−1​：第t-1期账面价值

会计关系

账面价值的更新：
​

Bt=Bt−1+NIt−DtB_t = B_{t-1} + NI_t - D_t Bt​=Bt−1​+NIt​−Dt​

即：期末账面价值 = 期初账面价值 + 净利润 - 股利

剩余收益模型的优点

  1. 关注经济利润而非会计利润
  2. 考虑了权益资本成本
  3. 不依赖于股利预测
  4. 与会计数据紧密结合

相对估值法

相对估值法概述

相对估值法是通过比较可比公司的估值倍数，来评估目标公司价值的方法。

核心思想：相似的公司应该有相似的估值

与绝对估值法的区别：

维度 绝对估值法 相对估值法
基础 公司自身现金流 市场可比公司
假设 未来可预测 市场有效
结果 内在价值 相对价值
主观性 高（预测多） 中（选可比公司）

主要估值倍数

PE（市盈率）

定义：股价与每股收益的比率

公式：

PE=PEPS\text{PE} = \frac{P}{\text{EPS}} PE=EPSP​

其中：

  • PPP：股价
  • EPSEPSEPS：每股收益

分类：

  • 静态PE：用过去12个月EPS
  • 动态PE：用预测未来12个月EPS
  • 滚动PE：用最近4个季度EPS

适用场景：

  • 盈利稳定的成熟公司
  • 消费、零售等行业

局限性：

  • 亏损公司无法使用
  • 受会计政策影响
  • 周期性公司不适用

PB（市净率）

定义：股价与每股净资产的比率

公式：

PB=PBPS\text{PB} = \frac{P}{\text{BPS}} PB=BPSP​

其中：

  • BPSBPSBPS：每股净资产（账面价值）

适用场景：

  • 银行、保险等金融行业
  • 重资产行业（钢铁、电力）
  • 亏损但资产优质的公司

局限性：

  • 轻资产公司不适用
  • 账面价值可能失真

PS（市销率）

定义：市值与营业收入的比率

公式：

PS=市值营业收入\text{PS} = \frac{\text{市值}}{\text{营业收入}} PS=营业收入市值​

适用场景：

  • 亏损但收入增长的公司
  • 初创企业、互联网公司
  • 周期性行业低谷期

局限性：

  • 不考虑盈利能力
  • 收入质量差异大

PEG（市盈率相对盈利增长比率）

定义：PE与盈利增长率的比值

公式：

PEG=PEg\text{PEG} = \frac{\text{PE}}{g} PEG=gPE​

其中：

  • ggg：盈利增长率（%）

判断标准：

  • PEG < 1：可能被低估
  • PEG = 1：合理估值
  • PEG > 1：可能被高估

适用场景：

  • 成长型公司
  • 需要考虑增长速度

局限性：

  • 增长率预测困难
  • 负增长无法使用

EV/EBITDA

定义：企业价值与息税折旧摊销前利润的比率

公式：

EV/EBITDA=EVEBITDA\text{EV/EBITDA} = \frac{\text{EV}}{\text{EBITDA}} EV/EBITDA=EBITDAEV​

其中：

  • EVEVEV：企业价值 = 市值 + 总债务 - 现金
  • EBITDAEBITDAEBITDA：息税折旧摊销前利润

为什么用EV/EBITDA？

优点 说明
不受资本结构影响 考虑债务，可比性更强
不受折旧政策影响 加回折旧摊销
不受税率影响 税前指标
可用于亏损公司 EBITDA通常为正

适用场景：

  • 资本密集型行业
  • 高负债公司
  • 跨国比较（税率不同）

可比公司分析

选择可比公司的原则

原则 说明
行业相同 主营业务相似
规模相近 收入、市值相当
商业模式相似 盈利模式、客户群体
成长性相当 增长率可比
地域相同 主要市场一致

分析步骤

  1. 筛选可比公司（通常3-10家）
  2. 收集估值倍数数据
  3. 计算可比公司平均/中位数倍数
  4. 调整差异（规模、成长性等）
  5. 应用到目标公司
  6. 敏感性分析

调整因素

因素 调整方向
规模更大 估值更高（流动性溢价）
增长更快 估值更高
盈利能力更强 估值更高
风险更高 估值更低

总结

倍数 公式 适用行业 优点 缺点
PE P/EPS 消费、零售、制造 直观易懂 亏损无效、会计影响
PB P/BPS 银行、保险、重资产 稳定可靠 轻资产无效
PS 市值/收入 互联网、初创、亏损 不受利润影响 不考虑盈利
PEG PE/g 成长股 考虑增长 增长率难测
EV/EBITDA EV/EBITDA 资本密集、高负债 可比性强 计算复杂

案例

第1步：找可比公司

公司 股价 每股收益 PE
喜茶 45元 2.5元 18倍
奈雪 18.5元 0.9元 20.6倍
蜜雪 120元 6元 20倍
茶百道 35元 1.8元 19.4倍

平均PE = 19.5倍

第2步：套用

你的奶茶店EPS = 0.30元

估值 = 19.5 × 0.30 = 5.85元/股
