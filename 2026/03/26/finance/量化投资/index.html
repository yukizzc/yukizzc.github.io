选股策略

GARP策略

  • 核心含义：GARP策略的核心是寻找兼具成长性与价值性的股票。它不像纯成长策略那样愿意为高增长支付任何价格，也不像纯价值策略那样只买便宜的烟蒂股。它追求的是用合理的价格买入成长。
  • 构建逻辑步骤：
    1. 筛选成长性：设定指标筛选出有稳定成长潜力的股票。常用指标：**预期盈利增长率、历史EPS（每股收益）增长率、ROE（净资产收益率）**等。
    2. 评估估值合理性：在上述股票池中，筛选估值相对合理的股票，剔除估值过高的标的。核心指标：PEG（市盈率相对盈利增长比率）**。通常寻找PEG在1附近或小于1的股票。
    3. 质量筛选（可选）：加入财务质量指标剔除财务造假或经营风险高的公司，如资产负债率、现金流状况。
    4. 打分与排序：将成长指标、估值指标、质量指标进行标准化打分，选出综合得分最高的股票组合。

因子模型

  • 核心含义：因子模型认为股票的收益不是随机的，而是可以被一系列共同的因子所解释和预测的。投资的过程变成暴露于能够带来收益的因子（如价值、规模），同时避开或对冲掉无用的风险。
  • 构建逻辑步骤：
    1. 因子挖掘与选择：寻找并验证能有效解释和预测股票收益的因子。常见大类：
      • 价值因子：低市盈率、低市净率。
      • 规模因子：小市值。
      • 动量因子：过去一段时间涨幅较高。
      • 质量因子：高ROE、低杠杆、盈利稳定性。
      • 波动率因子：低换手率、低波动。
    2. 因子测试：对单因子进行回测，检验其收益率、稳定性和换手率，剔除无效或衰减的因子。
    3. 因子合成：将选中的多个有效因子通过打分法或回归法进行合成，构建一个综合因子。
    4. 选股与调仓：在全体股票中，选择综合因子得分最高的N只股票构建组合，并定期（如每月）调仓。

打分法

这是最直观、最稳健、也是最常用的方法，特别适合散户和中小机构。

  • 核心逻辑：将所有股票按因子值排队，根据排名打分，最后将多个因子的分数加权求和
  • 操作步骤：
    1. 标准化处理：将不同量纲的因子值转换为排名百分位（0~100分）或Z-Score（均值为0，标准差为1）。
    2. 方向调整：确保所有因子方向一致。例如，市盈率是反向指标（越低越好），我们需要取它的倒数或者用负值，这样“得分越高”就代表“越好”。
    3. 权重设置：
      • 等权合成：各因子等权重相加，最经典、最不易过拟合
      • 动态权重：根据近期因子的ICIR（信息比率）表现动态调整
    4. 加总得分：得到每只股票的综合因子得分，然后选得分最高的前N只。
  • 优点：稳健，不受极端值影响
  • 缺点：丢失了因子值之间的“差距”信息

回归法

这是更偏计量经济学的方法，旨在精确拟合收益率。

  • 核心逻辑：通过线性回归计算因子对收益的贡献系数，然后用该系数预测未来收益

  • 操作步骤：

    1. 建立回归方程：

      Ri=α+β1⋅(Factor1i)+β2⋅(Factor2i)+ϵiR_i = \alpha + \beta_1 \cdot (Factor1_i) + \beta_2 \cdot (Factor2_i) + \epsilon_i Ri​=α+β1​⋅(Factor1i​)+β2​⋅(Factor2i​)+ϵi​

      其中，RiR_iRi​ 是股票i下个月的收益率，β1\beta_1β1​ 是因子1的系数（即因子暴露）。

    2. 历史回测：用过去3年或5年的数据，跑一遍回归，得到稳定的 β\betaβ 系数。

    3. 预测未来：假设 β\betaβ 系数稳定，将股票当前的因子值代入公式，计算出预期收益率 Ri^\hat{R_i}Ri​^​。

    4. 选股：选择预期收益率最高的股票。

  • 优点：理论上更精确，能处理因子间的相关性

  • 缺点：

    • 不稳定：对极端值非常敏感，一只股票的数据异常就可能改变整个回归系数。
    • 过拟合：容易把历史上的巧合噪声拟合成有效的系数，未来容易失效。

Alpha策略

  • 核心含义：Alpha策略追求的是剥离市场涨跌（Beta）影响后的绝对超额收益。即无论大盘涨跌，策略通过选股和对冲都能赚钱。在量化中通常指多因子选股 + 对冲（如做空股指期货）。
  • 构建逻辑步骤：
    1. 构建选股模型：通常利用多因子模型筛选出一篮子预期能跑赢大盘的股票组合（多头端）。
    2. 风险暴露控制：严格控制组合在风格因子（行业、市值、Beta）上的暴露，确保选股组合的超额收益主要来自于选股能力（纯粹的Alpha），而不是赌对了风格。
    3. 构建对冲端：做空等市值的股指期货（如沪深300、中证500期货），以对冲市场整体的系统性风险。
    4. 净值监控：组合的净值曲线主要体现为Alpha收益减去对冲成本，与大盘涨跌相关性低（甚至为负）。

风格轮动策略

  • 核心含义：金融市场存在不同的风格（如大盘/小盘、成长/价值）。风格轮动策略的核心是判断当下市场哪种风格占优，并将资金集中配置于占优风格，规避弱势风格。
  • 构建逻辑步骤：
    1. 定义风格：明确划分风格指数，如大盘成长、大盘价值、小盘成长、小盘价值。
    2. 寻找轮动指标（择时信号）：寻找能够预示风格切换的宏观指标或情绪指标。常见信号包括：
      • 宏观经济：PMI上行利好价值（周期股），利率下行利好成长。
      • 相对强弱指标：监控大小盘或成长价值的相对强度（如相对强弱曲线RSI），捕捉趋势。
      • 资金流向：观察主力资金流入流出哪个风格的板块。
    3. 构建信号模型：根据上述指标，生成当前该配置哪种风格的信号。
    4. 执行切换：根据信号，卖出弱势风格对应的ETF或股票池，买入强势风格的组合。

动量策略

  • 核心含义：也叫趋势跟踪策略。核心是 “强者恒强” 。认为过去一段时间表现好的股票，在未来一段时间内继续表现好的概率较大。
  • 构建逻辑步骤：
    1. 定义时间窗口：确定计算动量的周期。常见的有中期动量（6-12个月）和短期动量（频发交易）。通常剔除最近一个月（防止反转效应）。
    2. 计算动量指标：计算个股在过去N个月的累计收益率（或风险调整后的超额收益）。
    3. 排序与筛选：将所有股票按动量指标从高到低排序，选择排名前10%-20%的股票（赢家组合）。
    4. 风险控制：动量策略容易在牛市后期追高，在牛转熊时遭遇巨大回撤。通常需要结合止损纪律或低波动率筛选来辅助控制风险。

反转策略

  • 核心含义：与动量相反，核心是 “涨久必跌，跌久必涨” 。认为过去一段时间跌幅最深的股票，未来会触底反弹（超跌反弹）；或者过去涨幅过高的股票，未来会回落。
  • 构建逻辑步骤：
    1. 定义时间窗口：反转策略通常适用于短期（几天到几周，短线交易）或超长期（3-5年，均值回归）。A股市场中，短期反转效应（如月度）通常比长期反转更明显。
    2. 计算跌幅/涨幅：计算个股在过去一段时间（如过去20个交易日）的累计涨跌幅。
    3. 排序与筛选：将所有股票按涨跌幅从低到高排序，选择跌幅最深（排名倒数）的股票（输家组合）；或者卖出涨幅最高的股票。
    4. 基本面排雷：纯粹的反转会买到基本面崩盘的股票，因此通常会加入流动性筛选和基本面排雷（剔除ST股、剔除连续亏损股），确保买入的是被“错杀”的股票而非“垃圾股”。

投资书

聪明的投资者

格雷厄姆防御型投资者七大选股标准

筛选维度 具体量化标准 背后的投资逻辑
1. 企业规模 足够大：工业企业年销售额不低于1亿美元；公用事业总资产不低于5000万美元。 排除规模小的公司，因为它们抗风险能力弱，更容易在市场变化中被淘汰。
2. 财务状况 足够稳： • 流动比率（流动资产/流动负债）≥ 2。 • 长期债务不超过流动资产净额（营运资本）。 确保公司手头宽裕，没有沉重的债务负担，不会因为还不上钱而陷入危机。
3. 盈利稳定性 利润持续：过去10年中，每年都有盈利（没有亏损年份）。 证明公司经营稳健，有穿越经济周期的能力，不是“靠天吃饭”的。
4. 股息记录 分红长情：至少有20年连续支付股息的记录。 长期分红是公司财务健康、管理层对股东负责的“试金石”，也是投资者获得稳定现金流的保障。
5. 利润增长 基础增长：过去10年内，每股收益的增长至少达到三分之一（比较期初和期末的3年平均数）。 确保公司业务不是在走下坡路，能跟上甚至跑赢通胀，保持最基本的增长。
6. 市盈率 (PE) 不能太贵：当期股价 ≤ 过去3年平均利润的 15倍。 这是“安全边际”的体现。再好的公司，买贵了也很难赚钱。这个标准能帮你避开热门股的风险。
7. 市净率 (PB) 不能太高：当期股价 ≤ 最新报告资产账面值的 1.5倍。 组合拳：更精确的算法是 PE × PB ≤ 22.5 (即 15倍PE × 1.5倍PB)。 确保股价有实实在在的资产作支撑，防止为市场泡沫和无形资产支付过高溢价。
