基本语句

带返回值

Rust 是基于表达式的语言，代码块的最后一个表达式的值（省略分号）会被隐式返回。在 if、match 等分支结构中，所有分支的返回值类型必须保持一致。

循环

Rust 有三种循环：loop、while、for。

loop

loop 是 Rust 中的无限循环表达式，它允许通过 break 返回值。当需要退出循环时，break value 会跳出循环，并将 value 作为整个 loop 表达式的返回值。这使得 loop 既可以用于重复执行，也可以用于查找或等待某个结果

while

for

标签

Rust 中的循环（loop、while、for）可以添加标签（例如 'outer:），配合 break 'label value 语法，可以精确控制跳出哪个循环。这在处理嵌套循环时特别有用，可以跳出外层循环并返回结果，避免使用标志变量。

模式匹配

match 是 Rust 最强大的流程控制工具。

基本match

match 必须穷尽所有情况，如果有漏的话编译器也会提示。另外作为返回值也要求类型必须一致

解构

match 守卫

match 守卫（match guard）是 Rust 模式匹配的增强特性，通过在匹配模式后添加 if 条件（语法：pattern if condition =>），可以对匹配结果进行二次筛选。只有当模式匹配并且守卫条件为 true 时，才会执行该分支

绑定模式（@）

绑定模式（binding pattern）是 Rust 模式匹配中的一项特性，使用 @ 运算符（称为 “at 绑定”）将一个值绑定到变量，同时对这个值的内部结构进行匹配。这在需要同时使用整个值和其分解部分时特别有用，避免了重复计算或重新构造

if let 和 while let

用于简化只需要匹配一种模式的场景。

if let

while let
