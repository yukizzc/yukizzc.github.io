损益图

期权的四个基本头寸的损益结构：

头寸 损益特征 适用场景
买入看涨 亏损有限（权利金），收益无限 看大涨
卖出看涨 收益有限（权利金），亏损无限 不看涨
买入看跌 亏损有限（权利金），收益有限 看大跌
卖出看跌 收益有限（权利金），亏损有限 不看跌

期权平价公式

C+Ke−rT=P+S{\color{Red} C + Ke^{-rT} = P + S} C+Ke−rT=P+S

  • C：欧式看涨期权价格

  • P：欧式看跌期权价格

  • S：标的资产当前价格

  • K：行权价

  • r：无风险利率

  • T：剩余到期时间

  • e−rTe^{−rT}e−rT：连续复利折现因子

直观理解：

  • 左边：买入看涨期权 + 准备 Ke−rTKe^{-rT}Ke−rT 现金（未来自动增值到 KKK）
  • 右边：买入看跌期权 + 买入1股股票

到期时股价为 STS_TST​：

情况1：ST>KS_T > KST​>K（股价高于行权价）

  • 左边：看涨期权行权，以 KKK 买入股票，手中现金正好 KKK 元 → 最终持有 1股股票
  • 右边：看跌期权不行权，原本就有1股股票 → 最终持有 1股股票

情况2：ST<KS_T < KST​<K（股价低于行权价）

  • 左边：看涨期权不行权，手中现金 KKK 元 → 最终持有 现金 K
  • 右边：看跌期权行权，以 KKK 卖出手中股票，得到 KKK 元现金 → 最终持有 现金 K

结论：无论股价涨跌，两个组合到期价值完全相等，因此当前价值必然相等

BS公式

Black-Scholes公式是期权定价的基石，由Fischer Black和Myron Scholes于1973年提出，为此他们获得了诺贝尔经济学奖。

基本假设

  1. 标的资产价格服从几何布朗运动（对数正态分布）
  2. 无风险利率 rrr 为常数
  3. 波动率 σ\sigmaσ 为常数
  4. 市场无摩擦（无交易成本、无税收、可自由卖空）
  5. 期权为欧式（只能在到期日行权）

公式

欧式期权

C=S0N(d1)−Ke−rTN(d2)P=Ke−rTN(−d2)−S0N(−d1)C = S_{0}N(d_{1}) - Ke^{-rT}N(d_{2})\\ P = Ke^{-rT}N(-d_{2})-S_{0}N(-d_{1}) C=S0​N(d1​)−Ke−rTN(d2​)P=Ke−rTN(−d2​)−S0​N(−d1​)

其中

d1=ln⁡(S0K)+(r+σ2/2)TσTd2=ln⁡(S0K)+(r−σ2/2)TσT=d1−σTd_{1} = \frac{\ln(\frac{S_{0}}{K})+(r+\sigma^2/2)T}{\sigma\sqrt{T}}\\ d_{2} = \frac{\ln(\frac{S_{0}}{K})+(r-\sigma^2/2)T}{\sigma\sqrt{T}}=d_{1}-\sigma\sqrt{T} d1​=σT​ln(KS0​​)+(r+σ2/2)T​d2​=σT​ln(KS0​​)+(r−σ2/2)T​=d1​−σT​

符号 含义
S0S_0S0​ 标的资产当前价格
KKK 行权价
rrr 无风险利率（连续复利）
σ\sigmaσ 年化波动率
TTT 剩余到期时间（年）
N(⋅)N(\cdot)N(⋅) 标准正态分布累积函数CDF
N′(⋅)N'(\cdot)N′(⋅) 标准正态分布概率密度函数PDF
  • N(d2)N(d_2)N(d2​)：风险中性概率下，期权到期时被行权的概率
  • S0N(d1)S_0 N(d_1)S0​N(d1​)：期望收益的现值（当行权发生时）
  • Ke−rTN(d2)K e^{-rT} N(d_2)Ke−rTN(d2​)：行权成本的现值

公式本质：期权价格 = 期望收益的现值

BS公式用到的都是标准后的概率密度和分布函数

希腊字母

希腊字母是期权风险管理工具，衡量期权价格对不同因素的敏感度

Delta（Δ）

定义：期权价格对标的资产价格的一阶导数

Δ=∂V∂S\Delta = \frac{\partial V}{\partial S} Δ=∂S∂V​

公式：

  • 看涨期权：ΔC=N(d1)\Delta_C = N(d_1)ΔC​=N(d1​)
  • 看跌期权：ΔP=N(d1)−1\Delta_P = N(d_1) - 1ΔP​=N(d1​)−1

取值范围：

  • 看涨期权：[0,1][0, 1][0,1]
  • 看跌期权：[−1,0][-1, 0][−1,0]

核心含义：

  1. 方向敏感度：标的资产价格变动1单位，期权价格变动 Δ\DeltaΔ 单位
  2. 行权概率近似：Δ\DeltaΔ 近似等于期权到期时成为实值的概率（风险中性下）
  3. 对冲比例：要构建Delta中性组合，需要持有 −Δ-\Delta−Δ 份标的资产

特性：

  • 深度实值时 Δ→1\Delta \to 1Δ→1（看涨）或 Δ→−1\Delta \to -1Δ→−1（看跌）
  • 深度虚值时 Δ→0\Delta \to 0Δ→0
  • 平值时 Δ≈0.5\Delta \approx 0.5Δ≈0.5（看涨）或 Δ≈−0.5\Delta \approx -0.5Δ≈−0.5（看跌）

Gamma（Γ）

定义：Delta对标的资产价格的一阶导数（期权价格的二阶导数）

Γ=∂2V∂S2=∂Δ∂S\Gamma = \frac{\partial^2 V}{\partial S^2} = \frac{\partial \Delta}{\partial S} Γ=∂S2∂2V​=∂S∂Δ​

公式：

Γ=N′(d1)S0σT\Gamma = \frac{N'(d_1)}{S_0 \sigma \sqrt{T}} Γ=S0​σT​N′(d1​)​

其中 N′(d1)=12πe−d12/2N'(d_1) = \frac{1}{\sqrt{2\pi}} e^{-d_1^2/2}N′(d1​)=2π​1​e−d12​/2 是标准正态分布的概率密度函数

核心含义：

  1. Delta的变化速度：标的资产价格变动1单位，Delta变动 Γ\GammaΓ 单位
  2. 对冲难度：Γ\GammaΓ 越大，Delta变化越快，对冲越困难
  3. 曲率：Γ\GammaΓ 衡量期权价格与标的资产价格关系的弯曲程度

Gamma符号规律：

  • 期权买方：Γ>0\Gamma > 0Γ>0
  • 期权卖方：Γ<0\Gamma < 0Γ<0

Gamma分布特征：

  • 平值期权 Γ\GammaΓ 最大
  • 深度实值或深度虚值时 Γ\GammaΓ 趋近于0
  • 临近到期时，平值期权的 Γ\GammaΓ 急剧增大

物理类比（方便记忆）：

  • Delta = 速度
  • Gamma = 加速度
  • 买方Gamma为正 = 加速运动（Delta朝着1或-1加速）
  • 卖方Gamma为负 = 减速运动（Delta朝着0减速）

重要结论：

Gamma的绝对值越大，Delta向1或-1收敛的速度越快。买方希望Gamma大（波动大时获利多），卖方希望Gamma小（风险可控）。

Vega（ν）

定义：期权价格对隐含波动率的一阶导数

ν=∂V∂σ\nu = \frac{\partial V}{\partial \sigma} ν=∂σ∂V​

公式：

ν=S0TN′(d1)\nu = S_0 \sqrt{T} N'(d_1) ν=S0​T​N′(d1​)

核心含义：

  • 隐含波动率变动1%，期权价格变动的幅度
  • 衡量期权对波动率变化的敏感度

Vega符号：

  • 无论是看涨还是看跌，Vega都为正
  • 因为波动率上升，期权价格都会上涨（大涨大跌的可能性增加）

Vega分布特征：

  • 平值期权Vega最大
  • 深度实值或深度虚值时Vega小
  • 近月期权：Gamma大，但Vega小（时间短，波动率变化影响小）
  • 远月期权：Gamma小，但Vega大（时间长，波动率变化影响大）

重要结论：

做多波动率策略（如买入跨式）本质上是做多Vega，期望波动率上升。

Theta（Θ）

定义：期权价格对时间的一阶导数（时间衰减）

Θ=∂V∂t\Theta = \frac{\partial V}{\partial t} Θ=∂t∂V​

公式：

  • 看涨期权：ΘC=−S0N′(d1)σ2T−rKe−rTN(d2)\Theta_C = -\frac{S_0 N'(d_1)\sigma}{2\sqrt{T}} - rK e^{-rT} N(d_2)ΘC​=−2T​S0​N′(d1​)σ​−rKe−rTN(d2​)
  • 看跌期权：ΘP=−S0N′(d1)σ2T+rKe−rTN(−d2)\Theta_P = -\frac{S_0 N'(d_1)\sigma}{2\sqrt{T}} + rK e^{-rT} N(-d_2)ΘP​=−2T​S0​N′(d1​)σ​+rKe−rTN(−d2​)

核心含义：

  • 时间每流逝1天，期权价格衰减的幅度
  • 通常为负值（时间是买方的敌人，卖方的朋友）

Theta特征：

  • 平值期权Theta绝对值最大
  • 临近到期时，平值期权Theta急剧增大（时间价值加速衰减）
  • 深度实值或深度虚值时，Theta绝对值较小

直观理解：

  • 买方：每天支付时间价值，希望股价快速大幅波动
  • 卖方：每天赚取时间价值，希望股价平稳或小幅波动

Rho（ρ）

定义：期权价格对无风险利率的一阶导数

ρ=∂V∂r\rho = \frac{\partial V}{\partial r} ρ=∂r∂V​

公式：

  • 看涨期权：ρC=KTe−rTN(d2)\rho_C = K T e^{-rT} N(d_2)ρC​=KTe−rTN(d2​)
  • 看跌期权：ρP=−KTe−rTN(−d2)\rho_P = -K T e^{-rT} N(-d_2)ρP​=−KTe−rTN(−d2​)

核心含义：

  • 利率变动1%，期权价格变动的幅度
  • 对股票期权影响较小，对利率期权影响较大

Rho符号：

  • 看涨期权：ρ>0\rho > 0ρ>0（利率上升，看涨期权价格上升）
  • 看跌期权：ρ<0\rho < 0ρ<0（利率上升，看跌期权价格下降）

直观理解：

  • 利率上升时，买入股票占用资金的机会成本增加，资金更倾向于买入看涨期权（占用资金少），推高看涨期权价格
  • 利率上升时，资金倾向于卖出股票存入银行，看跌期权需求下降，价格下跌

年波动率=42%日波动率=42%∗1252月波动率=42%∗112年波动率 = 42\% \\ 日波动率 = 42\%*\sqrt{\frac{1}{252}} \\ 月波动率 = 42\%*\sqrt{\frac{1}{12}} \\ 年波动率=42%日波动率=42%∗2521​​月波动率=42%∗121​​

为什么是开根号？因为波动率是标准差，而方差随时间线性累加：Var(n天)=n×Var(1天)\text{Var}(n\text{天}) = n \times \text{Var}(1\text{天})Var(n天)=n×Var(1天)，所以 σ(n天)=σ(1天)×n\sigma(n\text{天}) = \sigma(1\text{天}) \times \sqrt{n}σ(n天)=σ(1天)×n​

波动率类型

类型 定义 特点
历史波动率 过去一段时间的实际波动率 客观，但滞后
隐含波动率 由期权价格反推的波动率 前瞻，反映市场预期
已实现波动率 未来实际发生的波动率 事后衡量

波动率特征

特征 含义 交易含义
序列相关性（聚集性） 高波动率后往往跟随高波动率 波动率具有惯性
均值回归 波动率会向长期均值回归 极端波动率后会收敛
动量效应 波动率上涨或下跌趋势可能延续 趋势策略在波动率上可能有效

波动率与期权价格

波动率越高，期权价格越高。原因：

  • BS公式假设价格服从对数正态分布
  • 波动率（σ\sigmaσ）是分布的标准差
  • σ\sigmaσ 越大，分布的尾部越厚
  • 大涨大跌的概率增加，期权价值上升

波动率微笑：相同标的、相同到期日的期权，虚值和实值期权的隐含波动率往往高于平值期权，形成“微笑”形状。

二叉树定价

二叉树是BS公式的离散化版本，更直观地展示了风险中性定价的核心思想。

单步二叉树原理

基本设定：

  • 当前股价：SSS
  • 未来两种可能：上涨到 SuS_uSu​，下跌到 SdS_dSd​
  • 上涨倍数：u=Su/Su = S_u / Su=Su​/S
  • 下跌倍数：d=Sd/Sd = S_d / Sd=Sd​/S

构建无风险组合：
持有 xxx 股股票，卖出1份看涨期权。组合价值在未来两种状态下相等：

Su⋅x−fu=Sd⋅x−fdS_u \cdot x - f_u = S_d \cdot x - f_d Su​⋅x−fu​=Sd​⋅x−fd​

解出对冲比例：

x=fu−fdSu−Sdx = \frac{f_u - f_d}{S_u - S_d} x=Su​−Sd​fu​−fd​​

这个 xxx 就是Delta的对冲比例。

无风险组合的收益：
组合的到期价值是确定的（无论股价涨跌都一样），因此应该以无风险利率贴现：

S⋅x−f=e−rT(Su⋅x−fu)S \cdot x - f = e^{-rT} (S_u \cdot x - f_u) S⋅x−f=e−rT(Su​⋅x−fu​)

代入 xxx 并整理：

f=e−rT[pfu+(1−p)fd]f = e^{-rT} \left[ p f_u + (1-p) f_d \right] f=e−rT[pfu​+(1−p)fd​]

其中风险中性概率为：

p=erT−du−dp = \frac{e^{rT} - d}{u - d} p=u−derT−d​

风险中性概率

ppp 被称为风险中性概率：

  • 它不是一个真实世界的概率，而是一个数学构造
  • 在风险中性世界中，所有资产的期望收益率都等于无风险利率
  • 用 ppp 计算的期望值，用无风险利率贴现，就得到当前价格

多步二叉树

将时间分成多个小步，重复上述过程：

  1. 从最后一步开始，计算每个节点的期权价值
  2. 用 f=e−rΔt[pfu+(1−p)fd]f = e^{-r\Delta t}[p f_u + (1-p)f_d]f=e−rΔt[pfu​+(1−p)fd​] 向前递推
  3. 最终得到初始时刻的期权价格

当步数足够多时，二叉树价格收敛于BS公式价格。

计算示例

已知条件：

  • 当前股价 S=20S = 20S=20
  • 上涨后 Su=22S_u = 22Su​=22，下跌后 Sd=18S_d = 18Sd​=18
  • 行权价 K=21K = 21K=21
  • 无风险利率 r=12r = 12%r=12
  • 到期时间 T=0.25T = 0.25T=0.25 年（3个月）

步骤1：计算 uuu 和 ddd

u=22/20=1.1d=18/20=0.9u=22/20=1.1\\ d=18/20=0.9 u=22/20=1.1d=18/20=0.9

步骤2：计算风险中性概率 ppp

erT=e0.12×0.25=e0.03≈1.03045p=erT−du−d=1.03045−0.91.1−0.9=0.130450.2=0.65225e^{rT} = e^{0.12 \times 0.25} = e^{0.03} \approx 1.03045\\ p = \frac{e^{rT} - d}{u - d} = \frac{1.03045 - 0.9}{1.1 - 0.9} = \frac{0.13045}{0.2} = 0.65225 erT=e0.12×0.25=e0.03≈1.03045p=u−derT−d​=1.1−0.91.03045−0.9​=0.20.13045​=0.65225

步骤3：计算期权到期价值

  • 股价上涨时：fu=max⁡(Su−K,0)=max⁡(22−21,0)=1f_u = \max(S_u - K, 0) = \max(22 - 21, 0) = 1fu​=max(Su​−K,0)=max(22−21,0)=1
  • 股价下跌时：fd=max⁡(Sd−K,0)=max⁡(18−21,0)=0f_d = \max(S_d - K, 0) = \max(18 - 21, 0) = 0fd​=max(Sd​−K,0)=max(18−21,0)=0

步骤4：贴现求当前价格

f=e−rT[pfu+(1−p)fd]=e−0.03×(0.65225×1+0.34775×0)f=0.97045×0.65225=0.633f = e^{-rT} \left[ p f_u + (1-p) f_d \right] = e^{-0.03} \times (0.65225 \times 1 + 0.34775 \times 0)\\ f = 0.97045 \times 0.65225 = 0.633 f=e−rT[pfu​+(1−p)fd​]=e−0.03×(0.65225×1+0.34775×0)f=0.97045×0.65225=0.633
