beautifulSoup

Created2026-03-29|Updated2026-03-29|computer
|Post Views:

BeautifulSoup 是一个可以从 HTML 或 XML 文件中提取数据的 Python 库。它能够将复杂的 HTML 文档转换成树形结构，并提供简单的方法来导航、搜索和修改这个结构。Beautiful Soup 4（简称 BS4）是目前的主流版本，支持 Python 3

安装与配置

BeautifulSoup 需要配合解析器使用，推荐使用 lxml：

解析器对比

解析器 使用方法 优点 缺点
Python标准库 BeautifulSoup(html, "html.parser") 内置无需安装 速度较慢，容错性一般
lxml HTML BeautifulSoup(html, "lxml") 速度快，容错性强 需要安装C库
lxml XML BeautifulSoup(html, "xml") 速度快，支持XML 需要安装C库
html5lib BeautifulSoup(html, "html5lib") 容错性最好，浏览器方式解析 速度慢

建议：优先使用 lxml 解析器。
