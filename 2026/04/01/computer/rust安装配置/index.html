Cargo 包管理器

什么是 Cargo

Cargo 是 Rust 的包管理器和构建系统，是 Rust 生态的核心工具。

核心功能：

  • 项目管理：创建、构建、运行、测试项目
  • 依赖管理：声明和下载第三方库
  • 版本管理：确保依赖版本兼容性
  • 发布管理：将包发布到 crates.io

在 Rust 开发中，几乎所有项目都使用 Cargo 进行管理，它让开发流程变得高效可靠。

Cargo 常用命令

基础命令

命令 说明
cargo new 创建新项目
cargo build 编译项目（调试模式）
cargo build --release 编译项目（发布模式，优化）
cargo run 编译并运行
cargo run --release 编译并运行（发布模式）
cargo check 检查代码（不生成二进制）
cargo test 运行测试
cargo doc 生成文档
cargo clean 清理 target 目录

命令详解

项目管理命令

命令 说明 示例
cargo update 更新所有依赖到最新兼容版本 cargo update
cargo clean 删除 target 目录 cargo clean
cargo search 在 crates.io 搜索库 cargo search serde
cargo add 添加依赖到当前项目 cargo add serde
cargo rm 从当前项目移除依赖 cargo rm serde
cargo install 安装二进制工具 cargo install ripgrep
cargo uninstall 卸载二进制工具 cargo uninstall ripgrep
cargo publish 发布包到 crates.io cargo publish
