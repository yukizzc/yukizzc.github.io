模块概述

std::net 模块提供了 Rust 标准库中的网络编程功能，主要包括：

  • TCP：面向连接的可靠字节流协议
  • UDP：无连接的不可靠数据报协议
  • IP 地址：IPv4、IPv6 地址处理
  • Socket 地址：IP 地址 + 端口号的组合

特点：

  • 同步 I/O 操作
  • 线程安全的网络类型
  • 与操作系统原生 socket 的轻量级封装

TCP 编程

  • 面向连接：需要三次握手建立连接
  • 可靠传输：保证数据顺序、完整性和重传
  • 流量控制：防止发送方淹没接收方
  • 字节流：无消息边界，需要应用层处理分包

TCP 协议特点

  • 字节流：无消息边界，需要应用层处理分包

TCP 关键方法说明

TcpListener：

  • bind() - 绑定地址开始监听
  • incoming() - 获取连接迭代器
  • accept() - 接受单个连接
  • set_nonblocking() - 设置非阻塞模式

TcpStream：

  • connect() - 连接到服务器
  • peer_addr() - 获取对端地址
  • local_addr() - 获取本地地址
  • shutdown() - 半关闭连接

UDP 编程

UDP 协议特点

  • 无连接：无需建立连接即可通信
  • 不可靠：不保证数据到达、顺序或去重
  • 数据报：有明确的消息边界
  • 低开销：比 TCP 头部小，延迟更低

数据序列化

在Rust中，TCP服务和客户端默认只能发送和接收字节流（u8数组）。为了发送复杂的数据结构（如结构体、枚举等），需要将这些数据结构进行序列化，然后在接收端将其反序列化回原始数据结构

serde 是一个通用的序列化和反序列化框架，具体的序列和反序列操作有底下这些具体的实现库来完成

  • serde_json 用于 JSON 格式
  • serde_yaml 用于 YAML 格式
  • serde_cbor 用于 CBOR 格式
  • serde_pickle 用于 Python 的 Pickle 格式
  • bincode 用于二进制序列化
