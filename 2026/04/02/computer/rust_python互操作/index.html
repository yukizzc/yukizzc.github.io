Created2026-04-02|Updated2026-04-02|computer
|Post Views:

FFI = Foreign Function Interface（外部函数接口）

PyO3 是 Rust 生态中与 Python 交互的首选工具，核心特性：

  • 双向调用：Rust 调用 Python，Python 调用 Rust
  • 类型安全：编译时检查类型转换，避免运行时错误
  • 零拷贝：支持直接内存访问，高性能数据传输
  • 异步支持：与 async/await 无缝集成
  • GIL 管理：自动处理全局解释器锁，避免死锁

适用场景：

  • 性能敏感模块用 Rust 重写
  • Python 调用现有 Rust 库
  • 混合语言项目开发
  • 计算密集型任务加速

Rust 写法 Python 写法 说明
func.call0() func() 无参数调用
func.call1((x,)) func(x) 1个参数
func.call2((x, y)) func(x, y) 2个参数
func.call((x,y,z), None) func(x,y,z) 多个参数
obj.call_method0("method") obj.method() 对象无参方法
obj.call_method1("method", (x,)) obj.method(x) 对象1个参数
obj.call_method2("method", (x,y)) obj.method(x,y) 对象2个参数
func.call((), Some(kwargs)) func(x=1, y=2) 关键字参数

双向调用实战

  • Rust 负责：执行具体任务（打印、计算），管理任务计数
  • Python 负责：决定任务是否执行（验证逻辑）
  • 双向交互：Rust 调用 Python 问"能执行吗？"，Python 调用 Rust 执行任务
